# Content Text Media

Content text media component supporting a combination of content and media assets.

- Label – _Optional field_

- Title – _Optional field_

  - The title will be set at a H2 level

- Rich Text Description

  - Content support limited to bold, italic, paragraph styling using the Richtext Component

- Button – _Optional field_

- Image or Video Configuration – The CTM requires either an image or video

- Reversed - position the image/video to the left.
